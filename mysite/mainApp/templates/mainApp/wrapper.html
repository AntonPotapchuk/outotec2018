    Datasets we are working with are fabric noise and motion data, the total size is 60 GB. We explore both sources of data to be able to predict fabric failure, based on current data from equipment. The main challenge is to find any kind of correlation between the data and event type (the process is stopped, the process is running, the failure). We will use different preprocessing techniques as well as statistical and machine learning methods to explore possible dependencies.
